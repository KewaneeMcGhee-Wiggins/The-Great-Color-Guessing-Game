# The Great RGB Guessing Game

A color-matching game: a secret RGB color is picked and shown as text, and six colored squares are on the board. Click the square that matches the secret color to win; wrong guesses turn a square dark gray. Hit "New Colors" to reshuffle and play again.

## Tech stack

- Vanilla JavaScript (DOM manipulation, event listeners)

## Running it

Open `The_Great_RGB_guessing_game.html` in a browser, or serve the folder:

```bash
npx serve .
```
